# LoomsLilly

LoomsLilly is a creative marketplace and community platform for arts, crafts, tutorials, events, discussions, and seller-driven commerce.

The platform is planned to support buyer accounts, seller accounts, shared buyer/seller accounts, product listings, cart and order flow, tutorials, community discussions, events, deals, search, and role-aware user experiences.
